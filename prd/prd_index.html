更新 SOP（点击展开）
  1. 手动更新：打开 prd_index.html，在页面底部脚本的 entries 数组中新增或修改条目，格式为 {title:'显示标题', path:'相对路径/index.html'}，保存并刷新页面。
  2. 自动扫描（快速列表）：在 `prd` 目录运行以下命令列出所有非模板的 index.html 路径： 
    find . -name index.html -not -path "./.agents/*"
  3. 生成 entries JSON（示例，需 Python）：在 `prd` 目录运行： 
    python3 - <<'PY'
import json,os
out=[]
for root,dirs,files in os.walk('.'):
  if '.agents' in root: continue
  if 'index.html' in files:
    path=os.path.join(root,'index.html').lstrip('./')
    title=path.rsplit('/',1)[0]
    out.append({"title":title,"path":path})
print('const entries = '+json.dumps(out,ensure_ascii=False,indent=2)+';')
PY
    将输出复制回 entries 变量位置即可完成批量更新。
  4. 提交与发布：更新完成后提交到 Git（例如 git add prd_index.html && git commit -m "update prd index"），并按项目既有发布流程推送。
