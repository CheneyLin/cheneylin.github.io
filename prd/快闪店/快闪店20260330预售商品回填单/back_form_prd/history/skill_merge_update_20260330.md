## 🎯 更新目标

将 H5 端前三个独立页面（商品选择、收货地址、小票信息）合并为一个完整的长页面，支持在手机模板框中上下滚动浏览，保持所有组件交互功能完整正常，并同步更新 PRD 文档。

### 核心变更点
1. **页面结构**: 三个独立页面 → 一个合并的长页面
2. **导航方式**: 标签页切换 → 上下滚动浏览
3. **用户体验**: 分步操作 → 一次性完整展示
4. **手机框架**: 固定高度 → 自适应高度（支持滚动）

##### b) 手机框架高度自适应
**变更前**: 固定高度 812px  
**变更后**: 最小高度 812px，自动扩展

```css
.phone-frame {
    width: 375px;
    height: auto;           /* 改为自适应 */
    min-height: 812px;      /* 最小高度 */
    overflow: hidden;       /* 隐藏溢出 */
}
```

##### c) 页面内容容器支持滚动
**变更前**: 固定高度，隐藏滚动条  
**变更后**: 自适应高度，底部留白

```css
.page-content {
    height: auto;                  /* 自适应高度 */
    min-height: calc(100% - 44px); /* 最小高度 */
    overflow-y: auto;              /* 允许垂直滚动 */
    padding-bottom: 100px;         /* 底部留白 */
}
```

##### d) 合并后的完整页面结构
