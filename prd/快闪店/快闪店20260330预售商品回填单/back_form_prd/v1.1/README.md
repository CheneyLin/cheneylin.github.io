# 快闪店预售商品回填单 - v1.1 版本说明

## 版本信息

- **版本号**: v1.1
- **发布日期**: 2026-03-30
- **兼容性**: 向后兼容 v1.0

## 目录结构

```
v1.1/
├── assets/              # 静态资源目录（图片、样式、脚本等）
├── prd_html/
│   └── prd_v1.1.html   # PRD 文档 HTML 版本
├── prd_md/
│   └── prd_v1.1.md     # PRD 文档 Markdown 版本
└── prototype_html/
    ├── admin_v1.1.html # 运营后台原型页面
    └── h5_v1.1.html    # H5 回填单原型页面
```

## 文件说明

### PRD 文档

- **prd_v1.1.html**: 可直接在浏览器中打开查看的 PRD 文档
- **prd_v1.1.md**: Markdown 格式的 PRD 文档，便于编辑和版本管理

### 原型文件

- **h5_v1.1.html**: H5 端回填单原型
  - 包含完整的商品选择、收货地址、小票信息流程
  - 支持分步骤表单导航
  - 响应式设计

- **admin_v1.1.html**: 运营后台原型
  - 活动管理
  - 回填单审核
  - 订单管理
  - 商品池管理

## 使用方法

### 方式一：通过主控页面访问（推荐）

1. 打开根目录下的 `index.html`
2. 使用页面顶部的版本选择器切换到 v1.1
3. 左侧查看 PRD 文档，右侧查看原型

### 方式二：直接访问文件

**PRD 文档**:
```bash
open ./prd_html/prd_v1.1.html
```

**H5 原型**:
```bash
open ./prototype_html/h5_v1.1.html
```

**运营后台原型**:
```bash
open ./prototype_html/admin_v1.1.html
```

## 与 v1.0 的区别

v1.1 在 v1.0 的基础上进行了架构升级：

1. **文件组织**: 采用版本化目录结构
2. **访问方式**: 支持通过 UI 切换版本
3. **兼容性**: 完全保留 v1.0 的所有功能
4. **扩展性**: 为未来版本迭代奠定基础

## 注意事项

1. **首次加载**: 首次打开大文件可能需要几秒加载时间
2. **浏览器缓存**: 如遇显示问题，请清除浏览器缓存
3. **本地访问**: 所有文件均为本地 HTML，无需服务器即可访问
4. **跨域限制**: 部分浏览器可能限制本地文件的跨域访问，建议使用主控页面

## 常见问题

### Q: 如何在 v1.0 和 v1.1 之间切换？
A: 打开 `index.html`，使用顶部的"PRD 版本"下拉框进行切换。

### Q: v1.1 和 v1.0 的内容有什么区别？
A: 当前 v1.1 是 v1.0 的完整克隆，内容相同。后续会在 v1.1 中添加新的业务需求。

### Q: 可以删除 v1.0 吗？
A: 建议保留 v1.0 作为历史版本参考和备份。

### Q: 如何预览修改后的原型？
A: 直接在浏览器中打开对应的 HTML 文件即可实时预览。
