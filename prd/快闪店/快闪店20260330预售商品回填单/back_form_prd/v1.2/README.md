## 文件说明

### PRD 文档

- **prd_v1.2.html**: 可直接在浏览器中打开查看的 PRD 文档
- **prd_v1.2.md**: Markdown 格式的 PRD 文档，便于编辑和版本管理

### 原型文件

- **h5_v1.2.html**: H5 端回填单完整原型
  - 商品选择与数量调整
  - 收货地址管理
  - 小票信息录入（支持图片上传）
  - 表单提交与验证
  - 成功弹层展示

- **admin_v1.2.html**: 运营后台完整原型
  - 线下场次管理（新增、编辑、配置）
  - 回填单审核流程
  - 订单管理
  - 商品池管理

- **app_v1.2.html**: App 端订单列表原型（v1.2 新增）
  - 展示快闪店订单列表
  - 支持订单状态筛选（全部、待支付、待发货、待收货、待评价）
  - 支持业务分类 Tab 切换（商城、快闪店、戳戳乐、云集市）
  - 订单卡片展示：商品信息、回填单号、物流信息（已发货状态）
  - 提供"售后服务"按钮
  - 支持下拉刷新和上拉加载更多
  - 空状态展示

## 使用方法

### 方式一：通过主控页面访问（推荐）

1. 打开根目录下的 `index.html`
2. 使用页面顶部的版本选择器切换到 v1.2
3. 左侧查看 PRD 文档，右侧查看原型

### 方式二：直接访问文件

**PRD 文档**:
```bash
open ./prd_html/prd_v1.2.html
```

**App 原型**:
```bash
open ./prototype_html/app_v1.2.html
```

## 与 v1.1 的区别

v1.2 在 v1.1 的基础上进行了功能扩展：

1. **新增功能**: App 端快闪店订单列表展示页
2. **设计参考**: 基于 IMG_3648.jpg 的设计风格
3. **交互优化**: 支持订单状态筛选和业务分类切换
4. **信息展示**: 完整商品列表和物流信息展示
5. **操作简化**: 仅提供"售后服务"按钮，移除详情页

## 注意事项

1. **首次加载**: 首次打开大文件可能需要几秒加载时间
2. **浏览器缓存**: 如遇显示问题，请清除浏览器缓存
3. **本地访问**: 所有文件均为本地 HTML，无需服务器即可访问
4. **跨域限制**: 部分浏览器可能限制本地文件的跨域访问，建议使用主控页面

## 常见问题

### Q: 如何在 v1.0、v1.1 和 v1.2 之间切换？
A: 打开 `index.html`，使用顶部的"PRD 版本"下拉框进行切换。

### Q: v1.2 新增了哪些功能？
A: 新增了 App 端快闪店订单列表展示页，用户可以在 App 中查看所有快闪店订单。

### Q: 为什么没有订单详情页？
A: v1.2 版本仅提供订单列表展示，详情页将在后续版本中实现。

### Q: 如何预览修改后的原型？
A: 直接在浏览器中打开对应的 HTML 文件即可实时预览。
