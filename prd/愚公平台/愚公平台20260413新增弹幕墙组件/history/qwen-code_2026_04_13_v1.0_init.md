   - app_bulletin_empty_state: 空状态提示
   - api_topic_posts: 话题帖子数据API
   - api_bulletin_config: 弹幕配置API

### 需求确认要点

- 数据来源：自有社区系统，话题ID拉取帖子
- 话题配置：手动输入多个话题ID（逗号分隔）
- 数据刷新：固定间隔30秒/60秒
- 弹幕展示：自动滚动，高密度，4种样式组合
- 排序方式：按点赞数/按最新发布时间
- 用户发送：不支持，仅展示话题帖子
- 空状态：友好提示信息
