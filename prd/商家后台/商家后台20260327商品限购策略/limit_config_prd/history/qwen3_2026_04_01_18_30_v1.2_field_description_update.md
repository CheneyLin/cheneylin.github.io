## 变更原因分析

### 优势
1. ✅ **专业性提升**: "SPU ID/SKU ID"是电商行业标准术语
2. ✅ **避免歧义**: "商品 ID"可能被误解为 SPU ID 或 SKU ID
3. ✅ **前后一致**: 与限购策略章节的"指定商品 ID"表述统一
4. ✅ **用户友好**: "指定 SPU/指定 SKU"更直观表达操作意图

### 可能的影响
1. ⚠️ **开发适配**: 前端代码注释需同步更新术语
2. ⚠️ **测试用例**: 测试文档中的字段描述需同步调整
3. ⚠️ **培训材料**: 运营手册需使用新的标准术语
