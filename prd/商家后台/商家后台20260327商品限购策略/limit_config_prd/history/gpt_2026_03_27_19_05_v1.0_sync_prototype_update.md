#### 变更原因

**PRD 依据**: Markdown PRD 第 68 行明确规定  
> "列表展示：策略 ID、人群包、商品范围（SPU/SKU）、加入条件、状态。"

**优化目标**:
1. **简化信息架构**: 移除冗余的"名称"字段，直接展示人群包标签（crowdLabel）
2. **提升可读性**: 减少列数，让用户聚焦核心信息
3. **保持一致性**: 与 HTML PRD 文档和 Markdown PRD 保持完全一致

## 用户体验优化效果

### 视觉改进
- **列数减少**: 从 7 列减少到 6 列（核心 5 列 + 辅助 1 列）
- **信息密度降低**: 每屏可显示更多有效信息

- **操作区域更宽敞**: 操作列有更多空间，按钮点击更精准
