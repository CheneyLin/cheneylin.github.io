##### JavaScript 交互增强
**新增监听逻辑**:
```javascript
// SKU/SPU类型切换监听
document.addEventListener('DOMContentLoaded', function() {
  const strategyTypeRadios = document.querySelectorAll('input[name="strategyType"]');
  const skuSelectorSection = document.querySelector('.sku-selector-section');
  
  strategyTypeRadios.forEach(radio => {
    radio.addEventListener('change', function() {
      if (document.querySelector('input[value="SKU"]').checked) {
        // SKU 限购：显示 SKU 列表勾选
        skuSelectorSection.style.display = 'block';
      } else {
        // SPU 限购：隐藏 SKU 列表
        skuSelectorSection.style.display = 'none';
      }
    });
  });
});
```

**功能说明**:
- 当选择"SKU 限购"时，自动显示 SKU 列表勾选界面
- 当选择"SPU 限购"时，隐藏 SKU 列表勾选界面
- 支持多个 SKU 复选框勾选
