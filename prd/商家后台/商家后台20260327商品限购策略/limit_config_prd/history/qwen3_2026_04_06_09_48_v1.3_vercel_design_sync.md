## 变更摘要

本次升级为 PRD 包应用 **Vercel 设计规范（Geist Design System）**，全面提升视觉一致性与专业度。核心变更包括：

### 1. 字体系统升级
- ✅ 引入 **Geist Sans** 作为主字体家族
- ✅ 实施严格的字重体系：400（正文）、500（UI）、600（标题）
- ✅ 应用负向字间距规范：
  - Display 48px: `-2.4px`
  - Heading 32px: `-1.28px`
  - Sub-heading 24px: `-0.96px`
  - Body 16px: `normal`

### 2. 色彩系统重构
- ✅ 主色调整为 Vercel Black (`#171717`) 替代纯黑
- ✅ 引入工作流色彩体系：
  - Ship Red: `#ff5b4f`（用于价格/警示）
  - Develop Blue: `#0a72ef`（用于交互高亮）
- ✅ 统一中性色阶：Gray 900 → Gray 50

### 3. 阴影边框技术全面应用
- ✅ 替换所有传统 CSS `border` 为阴影边框技术
- ✅ 标准边框阴影：`rgba(0, 0, 0, 0.08) 0px 0px 0px 1px`
- ✅ 卡片多层阴影栈：
  ```css
  box-shadow: 
    rgba(0,0,0,0.08) 0px 0px 0px 1px,      /* 边框层 */
    rgba(0,0,0,0.04) 0px 2px 2px,           /* 微抬升 */
    rgba(0,0,0,0.04) 0px 8px 8px -8px,      /* 环境深度 */
    #fafafa 0px 0px 0px 1px;                /* 内发光 */
  ```

### 4. 组件样式优化
- ✅ **按钮系统**：
  - Primary: `#171717` 背景 + 6px 圆角
  - Secondary: 白色背景 + 阴影边框
  - Pill Badge: `9999px` 全圆角 + 蓝色调背景
  
- ✅ **表单控件**：
  - Focus Ring: `hsla(212, 100%, 48%, 1)` 蓝色焦点环
  - 输入框边框采用阴影技术
  
- ✅ **徽章系统**：
  - 状态徽章使用 Pill 形状（`border-radius: 9999px`）
  - 成功态：蓝色调 `#ebf5ff` / `#0068d6`
  - 警告态：黄色调 `#fff3cd` / `#856404`

### 5. 布局与间距调整
- ✅ PRD 页面最大宽度从 1000px 扩展至 1200px
- ✅ 页面内边距从 24px 增加至 80px 40px（Gallery 留白哲学）
- ✅ 章节间距从 32px 增加至 80px（垂直呼吸感）

### 6. 交互体验增强
- ✅ 所有可交互元素添加平滑过渡（`transition: all 0.2s`）
- ✅ Hover 状态统一采用透明度变化或背景色微调
- ✅ 原型联动高亮效果改用蓝色焦点环风格
