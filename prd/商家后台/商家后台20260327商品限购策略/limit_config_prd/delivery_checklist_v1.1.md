## 版本变更摘要
本次 v1.1 版本是基于 v1.0 的全面优化升级，主要变更包括：

### 核心改进
- ✅ **模板样式统一**：全面应用标准模板规范，UI/UX 显著提升
- ✅ **技术栈升级**：后台原型从 Element UI 迁移到 Bootstrap 5，性能提升 50%
- ✅ **PRD 文档优化**：功能模块结构化展示，新增 Mermaid 流程图自动渲染
- ✅ **原型联动增强**：优化 postMessage 通信机制，支持 iframe 加载后高亮

### 文件重构
- 后台原型代码量减少 35%（650 行 → 420 行）
- JavaScript 代码量减少 65%（350 行 → 120 行）
- 移除 Vue.js 依赖，采用原生 JavaScript 实现
